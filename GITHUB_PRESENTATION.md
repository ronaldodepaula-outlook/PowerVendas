# Apresentação do Projeto — PowerVendas Gestão

Resumo rápido
-------------
Aplicação SaaS para gestão de vendas e assistência de baterias. Fornece módulos de CRM (clientes/veículos), controle de estoque, atendimentos com registro de garantias, financeiro e relatórios.

Por que este projeto é útil?
- Automatiza atendimento e controle de garantias
- Facilita auditoria de vendas e estoque
- Integração simples com APIs REST

Como testar rapidamente
-----------------------
1. Instale dependências: `npm install`
2. Rode em dev: `npm run dev`
3. Abra `/login` e use credenciais de teste (se houver)
